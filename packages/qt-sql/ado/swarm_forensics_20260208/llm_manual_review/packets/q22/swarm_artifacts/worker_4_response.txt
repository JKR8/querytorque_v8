```sql
WITH filtered_dates AS (
    SELECT d_date_sk
    FROM date_dim
    WHERE d_month_seq BETWEEN 1188 AND 1188 + 11
)
SELECT
    i_product_name,
    i_brand,
    i_class,
    i_category,
    AVG(inv_quantity_on_hand) AS qoh
FROM inventory
JOIN filtered_dates ON inv_date_sk = d_date_sk
JOIN item ON inv_item_sk = i_item_sk
GROUP BY
    ROLLUP (
        i_product_name,
        i_brand,
        i_class,
        i_category
    )
ORDER BY
    qoh,
    i_product_name,
    i_brand,
    i_class,
    i_category
LIMIT 100
```

**Changes**: Converted implicit joins to explicit JOINs and isolated the date filter into a CTE to push filtering early. This allows the optimizer to apply join reordering and reduces the working set before aggregation.

**Expected speedup**: 1.2-1.5x due to early date filtering and better join planning freedom.
