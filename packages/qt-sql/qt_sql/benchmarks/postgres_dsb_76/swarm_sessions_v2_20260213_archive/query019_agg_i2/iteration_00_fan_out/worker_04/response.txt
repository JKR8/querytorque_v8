## Modified Logic Tree

```
QUERY: (single statement)
├── [~] filtered_store [CTE]
│   └── SCAN store
├── [~] filtered_address [CTE]
│   └── FILTER ca_state = 'KS'
├── [!] zip_mismatch_pairs [CTE]
│   ├── JOIN (CROSS) filtered_store × filtered_address
│   └── FILTER substring(s_zip,1,5) <> substring(ca_zip,1,5)
├── [!] filtered_fact_chain [CTE]
│   ├── SCAN store_sales
│   ├── JOIN (INNER) date_dim ON d_date_sk = ss_sold_date_sk
│   │   └── FILTER d_year = 1999 AND d_moy = 2
│   ├── JOIN (INNER) item ON i_item_sk = ss_item_sk
│   │   └── FILTER i_category = 'Shoes'
│   ├── JOIN (INNER) customer ON c_customer_sk = ss_customer_sk
│   │   └── FILTER c_birth_month = 2
│   └── FILTER ss_wholesale_cost BETWEEN 76 AND 96
├── [~] join_with_pairs [CTE]
│   ├── JOIN (INNER) filtered_fact_chain × zip_mismatch_pairs
│   │   ├── ON ss_store_sk = s_store_sk
│   │   └── ON c_current_addr_sk = ca_address_sk
├── [~] aggregation [MAIN]
│   ├── AGG (GROUP BY i_brand, i_brand_id, i_manufact_id, i_manufact)
│   ├── SORT (ext_price DESC, i_brand ASC, i_brand_id ASC, i_manufact_id ASC, i_manufact ASC)
│   └── LIMIT 100
└── OUTPUT (brand_id, brand, i_manufact_id, i_manufact, ext_price)
```

**Changes:**
- Restructured into 6 CTEs matching target logical tree
- `zip_mismatch_pairs`: Early non-equi join reduction via MATERIALIZED CTE
- `filtered_fact_chain`: Star-join prefetching with selective filters pushed down
- All joins made explicit with INNER JOIN syntax
- Preserved all original semantics and output columns
