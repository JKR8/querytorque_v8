- output must be directly applicable PatchPlan JSON

## Critical Contract
- First character must be `{` (no leading whitespace/newlines).
- Return **ONLY** a single JSON object (no markdown fences, no prose).
- The JSON object must follow PatchPlan contract and include `steps`.

- Do not invent anchor hashes.

- Keep edits minimal and tied to the selected transform.
