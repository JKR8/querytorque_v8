# Prompt Versions

Consolidated prompt history by version.

## Layout
- `<version>/templates/` — template prompts (when available)
- `<version>/examples/` — concrete sample prompts
- `<version>/docs/` — version-specific docs/specs

## IN USE
- `V3`: current runtime prompt system
